# Assets Directory

This directory contains app icons and images.

For a production app, you would place:
- `icon.png` - App icon (1024x1024)
- `splash.png` - Splash screen image
- `adaptive-icon.png` - Android adaptive icon (1024x1024)
- `favicon.png` - Web favicon (48x48)

For now, these are placeholders. The app will use default Expo assets.
